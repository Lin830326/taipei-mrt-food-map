# 地圖不顯示診斷步驟

## 請按照以下步驟檢查:

### 步驟 1: 開啟開發者工具
1. 在瀏覽器中開啟 `index.html`
2. 按 **F12** 開啟開發者工具
3. 切換到 **Console (主控台)** 分頁

### 步驟 2: 查看錯誤訊息
請查找以下可能的錯誤:

#### 常見錯誤 1: API Key 問題
```
Google Maps API error: ApiNotActivatedMapError
```
**解決方式**: 需要在 Google Cloud Console 啟用 Maps JavaScript API

#### 常見錯誤 2: CONFIG 未定義
```
Uncaught ReferenceError: CONFIG is not defined
```
**解決方式**: `config.js` 檔案載入順序問題

#### 常見錯誤 3: 網域限制
```
Google Maps API error: RefererNotAllowedMapError
```
**解決方式**: API Key 有網域限制,需要允許 `localhost` 或 `127.0.0.1`

#### 常見錯誤 4: 計費未啟用
```
Google Maps API error: BillingNotEnabledMapError
```
**解決方式**: 需要在 Google Cloud Console 啟用計費帳戶

### 步驟 3: 檢查地圖容器
在 Console 執行:
```javascript
document.getElementById('map')
```
應該要顯示 `<div id="map">...</div>`,不是 `null`

### 步驟 4: 檢查 Google Maps API 載入
在 Console 執行:
```javascript
typeof google
```
應該要顯示 `"object"`,不是 `"undefined"`

### 步驟 5: 檢查 CONFIG 物件
在 Console 執行:
```javascript
CONFIG
```
應該要顯示設定物件,包含 `GOOGLE_API_KEY` 等屬性
